Name

GM_log — log messages to the JavaScript Console

Synopsis

function GM_log(message);
                     

                  

               

               
 
                  
                  
Description

                  
                  
GM_log places output in the JavaScript Console.  It is primarily used for debugging purposes.
                  

                  
               

               
 
                  
                  
History

                  
                  
GM_log was introduced in Greasemonkey 0.3.
